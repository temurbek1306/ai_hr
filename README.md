# AI HR

AI-powered Human Resources Management System

## Loyiha Haqida
Bu loyiha AI texnologiyalari yordamida HR jarayonlarini avtomatlashtirish uchun yaratilgan.

## Texnologiyalar
- Frontend: React + TypeScript + Vite
- Styling: Tailwind CSS
- Backend: (qo'shiladi)

## O'rnatish
```bash
# Loyihani klonlash
git clone https://github.com/temurbek1306/ai_hr.git

# Papkaga kirish
cd ai_hr

# Bog'liqliklarni o'rnatish
npm install

# Ishga tushirish
npm run dev
```
